# getFocusedTab (app profile manager)

Returns the tab object of the tab that is in focus.

## Syntax

`Microsoft.Apm.getFocusedSession().getFocusedTab();`

## Return value

The tab object.

## Example

```JavaScript
const tab = Microsoft.Apm.getFocusedSession().getFocusedTab();
tab.refresh();
```
